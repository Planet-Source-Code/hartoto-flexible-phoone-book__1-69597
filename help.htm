XB Super Data

XBasicPro 2006

 

SB Super Data adalah alat serbaguna. Dapat dijadikan menjadi berbagai aplikasi dengan mudah.

Misal: aplikasi daftar telepon, kamus, daftar buku dll.

Untuk mengubah program ini tidak perlu dikompilasi ulang, tetapi hanya dengan mengubah struktur data pada file access (data.mdb) dan file XML (config.xml).

 

Kunci untuk mengubah aplikasi ini menjadi berbagai macam aplikasi adalah terletak pada file database Acces (data.mdb) dan file XML (config.xml).

Anda hanya mengubah field-field yang sesuai pada file data.mdb dan disesuaikan dengan file config.xml

 

Misal saya aakan membuat aplikasi yang dapat menyimpan daftar telepon, alamat serta nomor HP dari relasi bisnis.
